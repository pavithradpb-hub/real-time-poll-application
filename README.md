# real-time-poll-application
## 📊 Real-Time Poll Application — Description  The Real-Time Poll Application is a simple web-based voting system that allows users to create polls, add multiple options, and vote instantly. Built using HTML, CSS, and JavaScript, it provides an interactive interface where results update immediately as users cast their votes.
 Each option displays a dynamic progress bar showing the percentage of total votes in real time. The application is lightweight, responsive, and does not require any backend or database, making it ideal for learning front-end interactivity and basic real-time UI behavior.
